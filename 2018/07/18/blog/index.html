HEXO блог

Пошаговая инструкция по установке блога на базе HEXO

Доказательства вообще менее служат для тех, кто хочет учиться, чем для тех, кто хочет спорить.
@Артур Шопенгауэр

Что есть Hexo?

Hexo-это быстрый, простой и мощный фреймворк блогов. Вы пишите посты в Markdown (или других языках) и Hexo генерирует статические файлы с красивой темой в считанные секунды.

Требования

Установка Hexo довольно проста. Однако, сперва, нужно установить пару вещей:

  • Node.js
  • Git

Если всё это уже есть, поздравляю! Просто установите Hexo с помощью npm:

$ npm install -g hexo-cli

После установки Hexo, запустите следующие команды в консоли для инициализации Hexo в папке .

$ hexo init <folder>
$ cd <folder>
$ npm install

После инициализации папки будут выглядеть так:

.
├── _config.yml
├── package.json
├── scaffolds
├── source
| ├── _drafts
| └── _posts
└── themes

Я не буду рассказывать вам как пользоваться HEXO. Возможно в другой статье. Приступим сразу к деплою.

Deploy

Для деплоя вам необходимо:

  1. Создать аккаунт на сайте www.github.com
  2. Создать репозиторий в этом аккаунте с названием своего логина и приставке .github.io. Например мой: “chifek.github.io”

  3. Далее в корне вашего блога запустите команду:

    $ npm install hexo-deployer-git --save

  4. Затем откройте файл _config.yml
    и там найдите строки замените след.: 

    # Deployment
## Docs:        https://hexo.io/docs/deployment.html
deploy:
type: git
repo: https://github.com/вашЛогин/вашЛогин.github.io.git

Если все прошло успешно, то попробуйте две команды:

hexo generate

hexo deploy

Затем впишите свой логин от GitHub’a и пароль.
И наслаждайтесь своим блогом.
Ссылка на ваш блог будет такой: вашлогин.github.io
